# Surya Eko Indrawan — Personal Website

Website pribadi portofolio **[suryaeko.github.io](https://suryaeko.github.io)**.

## Tech Stack

- **React 19** + **TypeScript** + **Vite**
- **Tailwind CSS v4** + **shadcn/ui** (neobrutalism style)
- **Motion** (page transitions & micro-animations)
- **Supabase** (database)
- **Vercel** (deployment)

## Made With

Proyek ini dibuat dengan **[Claude Code](https://claude.ai/code)** menggunakan model **deepseek-v4-pro**.

## Development

```bash
npm install
npm run dev      # http://localhost:5173
npm run build    # production build
```

## Environment

Copy `.env.example` ke `.env` dan isi environment variables:

```
VITE_SUPABASE_URL=https://your-project.supabase.co
VITE_SUPABASE_ANON_KEY=your-anon-key
```
